# MC Mod Swapper
 Quickly swap you mod folders with this small utility tool!

## How to install
 1. Download [the repository](https://github.com/LetGame/MC-Mod-Swapper/archive/refs/heads/main.zip).
 2. Unzip it.
 3. Launch `ModSwapper.py`.
 4. Enter 0 to add new folders.
 5. Enter name, folder name and Minecraft version for your mod pack.
 6. Insert mods into the `/files/[folder name]`.
 7. Launch it again.
 8. Enter the number of the mod pack to apply it.
 9. Enjoy!
